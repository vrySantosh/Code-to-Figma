📊 Data Flow

1
User selects component in VS Code editor
2
Parser extracts structure and styles → JSON schema
3
VS Code sends HTTP POST to bridge server
4
Bridge forwards via WebSocket to Figma plugin
5
Figma creates nodes from schema
6
Response flows back with Node ID
7
VS Code stores mapping for future syncs

Terminal 1: Start Bridge Server

Terminal 2: Launch Extension

cd "/Users/Santosh/Code to Figma/vscode-extension"
