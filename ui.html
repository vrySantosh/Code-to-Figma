How to Use

1 Extract Component Data: Use the extraction script to analyze your React Native or Web component. This generates a JSON representation of your UI with all styles and structure.
2 Copy JSON Output: The extractor will output a JSON object containing your component tree, styles, and metadata.
3 Select Target Device: Choose your target device dimensions or use custom sizes for web pages.
4 Import to Figma: Paste the JSON below and click Import. Your component will be created with full fidelity, editable text, and proper auto-layout.
💡 Pro Tips:
  • Works with React Native (View, Text, Image, ScrollView, FlatList, TextInput, etc.)
  • Supports Web HTML elements (div, span, button, input, img, etc.)
  • All text layers are editable, images are imported, SVG icons are vectorized
  • Flexbox layouts automatically convert to Figma Auto-layout
  • Use Cmd/Ctrl+Enter to quickly import JSON
