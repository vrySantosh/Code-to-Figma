# Plugin Description for Figma Community

Use this text when publishing to Figma Community.

## Short Description (60 chars max)
Import code components into editable Figma designs

## Full Description (500 chars max)

Transform your codebase into Figma designs instantly. Perfect for creating design documentation from code, showing designers what's implemented, and collaborating between dev and design teams.

✓ All RN components (View, Text, Image, ScrollView, FlatList, etc.)
✓ Complete style fidelity (colors, fonts, spacing, shadows)
✓ SVG icon import with vector editing
✓ Smart font loading & automatic fallbacks
✓ Flexbox → Auto-layout conversion

The only plugin that goes Code → Figma. Ideal for developers who code first.

## Use Cases

### 1. Documentation
**Problem:** Need design specs for an existing app
**Solution:** Import screens and components into Figma, share with stakeholders

### 2. Design Handoff
**Problem:** Designers want to see exactly what's implemented
**Solution:** Import current implementation, designers can compare and suggest improvements

### 3. Component Library
**Problem:** Need to document all reusable components
**Solution:** Extract all components, create visual component library in Figma

### 4. Client Presentations
**Problem:** Client wants to see designs, but you only have code
**Solution:** Import implementations into Figma, present as design mockups

### 5. Design System Migration
**Problem:** Moving from code-first to design-first workflow
**Solution:** Import existing components as starting point for new design system
