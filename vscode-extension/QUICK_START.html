🚀 Code to Figma - VS Code Extension Quick Start

Welcome! This guide will help you set up and start using the Code to Figma VS Code extension in minutes.

📦 Installation

Step 1: Install Dependencies

Open your terminal and navigate to the extension folder:

cd "/Users/Santosh/Code to Figma/vscode-extension"
npm install

Step 2: Compile TypeScript

npm run compile

Step 3: Install Bridge Server Dependencies

npm install express cors ws body-parser

⚙️ Configuration

Step 1: Get Figma Access Token

  1. Go to Figma Settings
  2. Scroll to "Personal Access Tokens"
  3. Click "Generate new token"
  4. Copy the token (you'll need it in VS Code)

Step 2: Get Figma File ID

Your Figma file URL looks like:

https://www.figma.com/file/ABC123XYZ/My-Design-File

Copy the part in bold (ABC123XYZ)

Step 3: Configure VS Code Settings

  1. Open VS Code Settings (Cmd/Ctrl + ,)
  2. Search for "Code to Figma"
  3. Set your Figma File ID
  4. Set your Figma Access Token

Or add to your settings.json:

{
  "codeToFigma.figmaFileId": "YOUR_FILE_ID",
  "codeToFigma.figmaAccessToken": "YOUR_TOKEN",
  "codeToFigma.outputFolder": "./figma-exports",
  "codeToFigma.autoSync": false
}

🎯 Usage

Step 1: Start Bridge Server

In a terminal window, run:

cd "/Users/Santosh/Code to Figma/vscode-extension"
node figma-bridge-server.js

You should see:

Figma Bridge Server running on http://localhost:3333
WebSocket server running on ws://localhost:3334
Waiting for Figma plugin to connect...

Keep this terminal open!

Step 2: Launch Extension in Development Mode

  1. Open vscode-extension folder in VS Code
  2. Press F5 (or Run → Start Debugging)
  3. A new VS Code window will open with the extension loaded

Step 3: Open Figma Plugin

  1. Open Figma (desktop or web)
  2. Open your design file
  3. Go to Plugins → Development → Code to Figma
  4. Look for "🟢 VS Code Connected" at the bottom
✅ Setup Complete! You're ready to start converting components!

💡 First Export

Example Component

Create a test file: test-component.tsx

import React from 'react';
import { View, Text } from 'react-native';

export const WelcomeCard = () => {
  return (
    <View style={{
      backgroundColor: '#FFFFFF',
      padding: 24,
      borderRadius: 16,
      width: 300
    }}>
      <Text style={{
        fontSize: 24,
        fontWeight: '700',
        color: '#000000'
      }}>
        Welcome!
      </Text>
      <Text style={{
        fontSize: 16,
        color: '#666666',
        marginTop: 8
      }}>
        This is a test component
      </Text>
    </View>
  );
};

Export to Figma

  1. Select the component code (or just place cursor inside)
  2. Right-click → "Code to Figma: Export to Figma"
  3. Or use status bar button "Export to Figma"
  4. Check Figma - your component should appear!

🔧 Commands

Command Description
Convert Component to JSON Parse component and save as JSON
Export to Figma Send component directly to Figma
Sync with Figma Update existing Figma node
Toggle Auto-Sync Enable/disable automatic syncing

🎨 Sidebar

Click the "Code to Figma" icon in the Activity Bar to see:

  • All synced components
  • Figma Node IDs
  • Last sync times
  • Quick file navigation

🔄 Auto-Sync

💡 Tip: Enable auto-sync to automatically update Figma when you save files!

Command Palette → "Code to Figma: Toggle Auto-Sync"

❓ Troubleshooting

Bridge Server Not Connecting

Check if port 3333 is available:

lsof -i :3333

If blocked, kill the process or change the port in figma-bridge-server.js

Figma Plugin Shows Disconnected

  1. Ensure bridge server is running
  2. Restart the Figma plugin
  3. Check firewall settings

Extension Not Loading

  1. Compile TypeScript: npm run compile
  2. Restart VS Code Extension Host (F5)
  3. Check Output panel for errors
